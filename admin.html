Verlanglijst Beheer

Handleiding

  1. Je kunt hier items toevoegen aan je verlanglijst
  2. Vul de naam, URL (waar het te koop is) en optioneel een afbeelding URL in
  3. De wijzigingen worden direct zichtbaar voor bezoekers
  4. Je kunt items verwijderen met de "Verwijder" knop
  5. Alle wijzigingen worden automatisch opgeslagen in GitHub
⚠️ Let op: Je moet ingelogd zijn op GitHub om wijzigingen te kunnen maken.
